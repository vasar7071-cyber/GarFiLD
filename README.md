SetHome — uMod/Oxide plugin для Rust

Описание
- Даёт игрокам возможность ставить метки дома и телепортироваться к ним.

Команды
- /sethome [name] — установить дом (по умолчанию `home`).
- /home [name] — телепортироваться к дому.
- /delhome [name] — удалить дом.
- /homes — показать список собственных домов.

Установка
1. Скопируйте `SetHome.cs` в папку `oxide/plugins` на вашем сервере Rust (uMod/Oxide).
2. Перезапустите или выполните `oxide.reload SetHome`.

Примечания
- Плагин сохраняет данные через DataFileSystem Oxide; файл данных появится в `oxide/data/SetHome.json`.
- По умолчанию нет отдельных прав/разрешений — все игроки могут пользоваться командами. При необходимости можно добавить проверку permission.
